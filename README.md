# Ticket Rising Project

A web-based ticket raising (helpdesk) system designed for company internal use.

## Features
- Works in Offline (Local Network)
- Works in Online (Internet)
- Centralized ticket management
- Role-based access (User, Admin, Technician)

## Tech Stack
- Backend: Python (Flask)
- Frontend: HTML, CSS, JavaScript
- Database: MySQL (Local Network)

## Usage
This project is designed to be used inside a company to collect, track, and resolve IT support tickets.
